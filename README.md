#  Loan Default Risk Analysis

##  Project Overview
This project analyzes loan data to understand borrower behavior and identify key factors that influence loan default risk.

The analysis includes data cleaning, feature engineering, visualization, and advanced statistical techniques.

---

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

---

##  Key Analysis Performed
- Data Cleaning and Preprocessing
- Feature Engineering (Loan-to-Income Ratio)
- Data Visualization (Bar Charts, Scatter Plots, Heatmaps)
- Crosstab and Percentage Analysis
- Correlation Analysis
- Groupby Analysis

---

##  Key Insights
- Loan-to-income ratio is the strongest predictor of default risk
- Loan grade effectively reflects borrower risk
- Larger loans are associated with higher default probability
- Income alone is not sufficient to predict repayment behavior
